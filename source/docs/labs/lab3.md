# Лабораторная 1. CI/CD для статического сайта в SourceCraft.

## Цель

Освоить настройку CI/CD для статического сайта в SourceCraft/GitHub.

## Задание

1. Добавить дополнительный удаленный репозиторий SouceCraft.
2. Создать CI/CD настройку (ci.yaml для SouceCraft, workflow для GitHub)
3. Отправить код в удаленные репозитории

## Результат

По любому пушу код будет автоматически собираться и развертываться в удаленные репозитории в relase ветку.
