# Генератор бинарных деревьев

Лабораторная работа №5: "Построение бинарного дерева"

## Описание
Программа на Python для нерекурсивного построения бинарного дерева.
Каждый узел дерева имеет двух потомков, которые вычисляются по заданным формулам.

## Вариант 18
- **Корень (root):** 18
- **Высота (height):** 5
- **Левый потомок:** `(родитель - 8) * 3`
- **Правый потомок:** `(родитель + 8) * 2`

## Структура проекта
- `binary_tree.py` - основная реализация функции `gen_bin_tree`
- `test_binary_tree.py` - unit-тесты
- `README.md` - документация

## Использование
```python
from binary_tree import gen_bin_tree

# Создание дерева с параметрами по умолчанию
tree = gen_bin_tree()

# Создание дерева с пользовательскими параметрами
tree = gen_bin_tree(
    height=3,
    root=10,
    left_branch=lambda x: x * 2,
    right_branch=lambda x: x + 5
)
