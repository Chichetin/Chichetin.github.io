# Лабораторная 1. Создание статического сайта.

## Цель

Освоить создание статического сайта с использованием **MkDocs** и **GitHub Pages**.

## Задание

1. Создать репозиторий на GitHub.
2. Установить **MkDocs**.
3. Создать статический сайт.
4. Разместить сайт на GitHub Pages.

## Результат

Сайт создан и размещен на GitHub Pages.
